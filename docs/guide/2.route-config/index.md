# 路由配置

路由配置是 vitarx-router 的核心，它定义了 URL 路径与组件之间的映射关系。通过灵活的路由配置，你可以构建从简单到复杂的各种页面导航结构。

本章将详细介绍如何定义和配置路由，包括基础路由、动态参数、嵌套结构、命名路由、重定向与别名，以及路由元数据。

- [动态路由](./2.dynamic-routes.md) — 使用动态参数匹配灵活的路径
- [嵌套路由](./3.nested-routes.md) — 构建多层级的路由结构
- [命名路由与命名视图](./4.named-routes-views.md) — 通过名称引用路由与多视图渲染
- [重定向与别名](./5.redirect-alias.md) — 路由重定向与路径别名
- [路由元数据](./6.route-meta.md) — 为路由附加自定义数据
