# 安装

## 前置条件

vitarx-router 依赖 Vitarx 框架，请确保项目中已安装 `vitarx` v4.0.0 及以上版本：

```bash
# 查看 vitarx 版本
npm list vitarx
```

::: warning
如果你的项目尚未安装 Vitarx，请先参考 Vitarx 官方文档完成框架的安装与初始化。
:::

## 安装 vitarx-router

使用你喜欢的包管理器安装 `vitarx-router`：

```bash [npm]
npm install vitarx-router
```

```bash [pnpm]
pnpm add vitarx-router
```

```bash [yarn]
yarn add vitarx-router
```
